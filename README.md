Например, у нас есть:

- Два url: /account и /catalog
- Виджет: ава+фио (<UserBlock />) - используется и в /account, и в /catalog
- Каталог - состоящий из карточек и контрол-кнопок
- Вспомогательные функции (getTranslateName и isOnlineUser)

    - user-block
      - user-block.tsx (ui компонент - используется в двух роутах)
      - user-block.model.ts (лежат ютилити функции, конкретные для этой фичи, например, isOnlineUser.ts)

- pages

  - catalog
